關於Lion

我是賴彥谷 Lion ，興趣是繪畫、騎摩托車，目標是成為軟體工程師。

大學時期和社團朋友約好開店的夢想，從餐飲工作出發，一直到合夥開飲料店，發現經營店家是個全面的學問。
付出人生的學費後，繼續摸索的道路上，自學平面設計、程式語言課程…等，我認為能在而立之年能找到目標感到無比幸運。

雖然程式能力這條路上需要更努力，但跨領域反而是優勢。

我理解跨領域的盲點，解釋給非資訊部門或客人更具耐心，更能換位思考，盡快搭起雙方橋樑使溝通更為和諧，
而現今需要的是合作的力量，透過跨領域去碰撞新火花，也因如此，挑戰未知也容易把心態歸零。

因為這樣的經歷，我有自學解決問題的能力，理解問題核心進行拆解，盡量善用網路或政府資源學習精進，保持獨立思考。

我仍然還記得社團的宗旨：
「我樂於付出，只因我總在接受，我祈求世間充滿溫暖，只因人生本當如此」。

我的作品集

新北市職訓局 Java App程式設計實戰班 / 筆記本應用程式

在為期三個月的密集訓練課程，完成自己的專案成果，我非常喜歡用keep記事， 因為乾淨簡約而且隨時開啟，所以結業專案就是以google Keep為藍圖完成的HodaApp， 也有上架到GooglePlay商店，雖然沒人下載(笑)，但我也往程式之路邁進。

自行接案

開店期間結識許多店家老闆，因此獲得不少的設計練習，一開始先以菜單、海報為嘗試 ，後來也有製作商品LOGO跟大型刊物的機會。

我認為與客戶的對談是個很好的經驗，從客戶給出的「感覺」去推論出「視覺」， 就像在漫天的想法泡泡捕捉到枝微末節，一度也讓我困擾許久。

後來也找出解決之道，讓客戶運用google的圖片搜尋，反覆咀嚼後，思考該朝哪個方向前進， 透過不斷交流反饋，可以更快抓到客戶的具體想法，也大大減少來往修改的時間。

文創人物 / Hodala 猴達拉樂團

猴達拉取自「乾杯」的台語音譯，是自行創作的虛擬人物，以前常用「猴囝仔」去稱呼頑皮的孩子， 代表孩子純真好奇、青春無畏，而頭戴帽子、身著披風、手持樂器就像中古世紀的吟遊詩人， 走遍四方，貼近人生，訴說著一段又一段故事。

Line貼圖創作

工作閒暇之餘，腦袋都會著構思人物設定，便開始動筆著手嘗試Line貼圖， 藉由雙手讓人物鮮活是個很有趣的事情，也因如此讓我對看事的角度變得幽默， 性格上較顯得內斂，而每經歷一段歷程都會有新的體悟，也賦予新的靈感。

Facebook專頁 / Hodala吟遊生活

有時故事令人著迷，可能是因為幻想，卻又貼近真實，圓了生活的缺憾。

原本是單純的想法，希望以架空的虛擬人物角度來經營，當投入時會發現需要題材收集， 頻發文章才能取得關注度，朋友的建議...等，慢慢的跟經營實體店面相似，靜心思考後，果然還是要學習呢(笑)。

文創小物

開店期間因緣巧合有機會製作周邊商品，期間認識一位厲害的設計者，因此學習許多印刷的知識， 出來的成果受到許多支持，雖然獲得成就感著實令人開心，從無到有的經驗才是最寶貴的。

未來展望

想跨領域的緣故，接觸到職訓局java訓練班，而那時上課地點恰巧安排在台科大，重返校園對於我來說非常有啟發性， 可能被大學的書卷氣息感染，亦或是我也迫切想改變的原因，那樣的契機激發我對學習的熱忱，至今也仍是如此。

喜歡圖文創作的我，思考著能結合程式語言的話真是太棒了，學習越多，也開始描繪著心中藍圖。

「if I can do something for ...」

種子在我腦中開始成長，我想要紀錄著什麼，紀錄上個世代的敦敦教誨，收藏曾活過著證明，在這個過渡提倡做自己的世代，再微調回來一些些， 如果我能透過應用程式，網站去達成這些目標，隨著時光過去後，如果能把這些「知識」好好傳承到下個世代，那真的很棒呢。
